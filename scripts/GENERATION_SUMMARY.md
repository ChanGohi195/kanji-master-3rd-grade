## 例文の特徴

### 1. 小学2年生向け
- 簡単な語彙を使用
- 短くて分かりやすい文
- 日常生活に関連した内容

### 2. 読み方
- 各例文に対象漢字の読み方を明記
- 音読み（on）と訓読み（kun）を区別
- 既存の読みデータ（kanji-list.json）を活用

### 3. フォーマット
各例文には以下の情報を含む：
- `id`: 例文ID（例: U+4E38-ex-3）
- `sentence`: 例文（対象漢字を含む）
- `reading`: 対象漢字の読み
- `type`: 読みのタイプ（kun/on）
- `sentenceWithRuby`: ルビ付き例文（対象漢字以外にルビ）
- `sentenceHiragana`: 全ひらがな版

## 例文サンプル

### 丸（まる）- kun読み
- **例文**: 丸い ボール。
- **ルビ付き**: 丸い ボール。
- **ひらがな**: まるい ボール。

### 星（ほし/せい）
1. **kun読み**: きれいな 星。→ きれいな ほし。
2. **on読み**: わく星を みる。→ わくせいを みる。
3. **kun読み**: 星が ひかる。→ ほしが ひかる。

## 生成スクリプト

### 主要スクリプト
1. `generate-batches-3-16.js` - メイン生成スクリプト
2. `add-missing-examples.js` - 不足漢字の追加
3. `add-missing-kanji-149.js` - 漢字149「鳴」の追加
4. `fix-hiragana-errors.js` - ひらがな変換エラーの修正
5. `final-cleanup.js` - 最終クリーンアップ
6. `verify-batches.js` - 検証スクリプト

### 実行コマンド
```bash
cd scripts
node generate-batches-3-16.js
node add-missing-examples.js
node add-missing-kanji-149.js
node fix-hiragana-errors.js
node final-cleanup.js
node verify-batches.js
```
